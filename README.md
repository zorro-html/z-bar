# `<z-bar>`

Bar element which often positioned on the top of a page

`<z-bar-minor>` element could be the sub area on the right to contains menus, links etc.

## Import

```
<link rel="import" href="z-bar/z-bar.html">
```

## Examples

```
<z-bar>
  <h1>Hello World</h1>
  <z-bar-minor>
    <z-btn>login</z-btn>
    <z-btn>help</z-btn>
    <z-btn>feedback</z-btn>
  </z-bar-minor>
</z-bar>
```
